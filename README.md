# 动物森林 - Animal Forest

儿童教育游戏 MVP

## 运行方式

直接用浏览器打开 `index.html` 即可运行，无需服务器。

## 游戏特性

### 幼儿模式 (3-6岁)
- 🦊 **小狐狸** - 认数字 (1-20)
- 🐻 **小熊** - 认图形 (圆形/方形/三角形等)
- 🐦 **小鸟** - 认颜色

### 儿童模式 (7-12岁)
- 🐱 **小猫** - 认汉字
- 🐵 **小猴** - 加减乘除
- 🦉 **小猫头鹰** - 逻辑排序

## 功能

- ⭐ 星星奖励系统
- 🏆 3个难度关卡
- 📊 进度追踪
- 👨‍👩‍👧 家长面板 (PIN保护)
- 🔊 音效系统
- 📱 响应式布局

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API
- localStorage

## 开发

无外部依赖，直接打开 `index.html` 即可。
